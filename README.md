# neobis-week3

## Project Setup
```bash
git clone git@github.com:Aktan-A/neobis-week3.git
./mvnw clean spring-boot:run
```

## Running Tests
```bash
./mvnw test
```

## Technologies
- Spring
- Spring MVC
- MySQL
- Lombok
- JWT

## Environment Variables
| Key         | Description               |
|-------------|---------------------------|
| DB_URL      | Database url.             |
| DB_USERNAME | Database user username.   |
| DB_PASSWORD | Database user password.   |
| SECRET_KEY  | Secret for access tokens. |
